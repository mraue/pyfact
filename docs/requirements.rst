===================================
Requirements
===================================

PyFACT is written for Python 2.6 or higher and requires the following python packages to be installed:

numpy 1.5.1 (or higher)
    http://numpy.scipy.org/
scipy 0.9.0 (or higher)
    http://www.scipy.org/
pyfits 3.0.0 (or higher) 
    http://www.stsci.edu/resources/software\_hardware/pyfits/

The data needs to be in CTA eventlist FITS format v1.0.0 or higher.

Optional, for plotting:

matplotlib 1.0.1 (or higher)
   http://matplotlib.sourceforge.net/
